Install instructions
====================

1. Extract this zip.
2. Copy all folders inside:
   daffodil_smart_city/

3. Paste them into your existing XAMPP project folder:
   C:\xampp\htdocs\diu_electricity_monitoring\daffodil_smart_city\

After copy, one example should look like:
   C:\xampp\htdocs\diu_electricity_monitoring\daffodil_smart_city\knowledge_tower\index.html
   C:\xampp\htdocs\diu_electricity_monitoring\daffodil_smart_city\knowledge_tower\assets\css\style.css
   C:\xampp\htdocs\diu_electricity_monitoring\daffodil_smart_city\knowledge_tower\assets\js\app.js

Open example:
   http://localhost/diu_electricity_monitoring/daffodil_smart_city/knowledge_tower/

Important:
   The pages already include frontend structure and API connection logic.
   For device cards to show, your admin/backend device data must use matching building names.
